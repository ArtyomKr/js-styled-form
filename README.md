# Installation

Clone this repo:

```
git clone https://github.com/ArtyomKr/js-styled-form.git
```

Go to project folder:

```
cd ./path-to-project
```

Install dependencies:

```
npm i
```

Run the development server:

```
npm run server
```

Run webpack:

```
npm run serve
```
